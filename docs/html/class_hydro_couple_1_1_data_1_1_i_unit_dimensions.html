virtual double HydroCouple::Data::IUnitDimensions::getPower ( FundamentalUnitDimension  dimension) const
pure virtual

Returns the power for the requested dimension.

Parameters
dimensionrepresents the fundamental unit

For a quantity such as flow, which may have the unit m3/s, the getPower method must work as follows:

myDimension.getPower(FundamentalDimension::AmountOfSubstance) –> returns 0

myDimension.getPower(FundamentalDimension::Currency) –> returns 0

myDimension.getPower(FundamentalDimension::ElectricCurrent) –> returns 0

myDimension.getPower(FundamentalDimension::Length) –> returns 3

myDimension.getPower(FundamentalDimension::LuminousIntensity) –> returns 0

myDimension.getPower(FundamentalDimension::Mass) –> returns 0

myDimension.getPower(FundamentalDimension::Temperature) –> returns 0

myDimension.getPower(FundamentalDimension::Time) –> returns -1
