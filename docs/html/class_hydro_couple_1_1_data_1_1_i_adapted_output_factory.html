Detailed Description

IAdaptedOutputFactory is used to create instances of IAdaptedProducerExchangeItems .

This class can be internal to an IModelComponent by calling IModel::adaptedProducterExchangeItemFactories() or can be generated from an HydroCouple::IAdaptedOutputFactoryComponent;
